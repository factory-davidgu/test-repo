Complete setup instructions for GitHub Actions integration

1Obtain Your API Key

First, you need to get your Factory AI API key from the Factory AI platform:

  1. Visit Factory AI: Navigate to app.factory.ai
  2. Sign in or Create Account: Log in to your Factory AI account. If you don't have one, you'll need to create an account first.
  3. Access API Settings: Once logged in, go to your account settings or API settings page to find your API key.
  4. Generate or Copy Key: Generate a new API key if needed, or copy your existing API key. Keep this key secure!
üîí Security Note: Your API key is like a password. Never share it publicly or commit it to your code. Always store it as a GitHub secret.

2Add Key to GitHub Repository

Now that you have your API key, you need to add it to your GitHub repository as a secret:

  1. Navigate to Repository Settings:
    Go to your GitHub repository and click on the Settings tab at the top of the page.
  2. Access Secrets and Variables:
    In the left sidebar, find the Security section and click on Secrets and variables, then click on Actions.
  3. Create New Secret:
    Click the New repository secret button (green button in the top right).
  4. Enter Secret Details:
    In the form that appears:
    • Name: Enter exactly FACTORY_API_KEY
    • Secret: Paste your Factory AI API key that you copied in Step 1
  5. Save the Secret:
    Click the Add secret button to save. Your API key is now securely stored!
Secret Name: FACTORY_API_KEY
Secret Value: [Your API key from app.factory.ai]

3Verify Setup

After adding the secret, your GitHub Actions workflows will automatically use it. To verify:

  1. Create a new pull request or push changes to trigger the workflows
  2. Navigate to the Actions tab in your repository
  3. Check that the Droid CLI workflows run successfully without authentication errors
  4. If you see errors about missing API keys, double-check that the secret name matches exactly
